About This Project

The Mango Pest & Disease Guide is a comprehensive resource designed to help mango growers, agriculturists, and students identify and manage common pests and diseases affecting mango crops in Australia.

This project was developed as part of a university course assignment, aimed at creating a practical tool for the agricultural community while demonstrating web development skills.

Project Information

This website was built using Django, a high-level Python web framework, along with HTML, CSS, and JavaScript. The database contains information about common mango pests and diseases, their characteristics, and management strategies.

All information provided has been researched and compiled from reliable agricultural resources and scientific literature. For a complete list of references, please visit our References page.
